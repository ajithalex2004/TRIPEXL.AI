Explanation

The CSS selector .gm-style-iw-c button.gm-ui-hover-effect targets the close button in Google Maps InfoWindow components.

By setting display: none !important, we ensure the button is always hidden, regardless of other styles.

In the example above, the second InfoWindow has the Google Maps classes applied, demonstrating how our CSS works.
